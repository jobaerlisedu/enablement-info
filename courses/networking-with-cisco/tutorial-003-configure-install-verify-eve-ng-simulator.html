Configure, Install & Verify EVE-NG Simulator

12 May, 2026 Networking Concepts Jobaer Hossain

আপনার ল্যাব এনভায়রনমেন্ট রেডি করার জন্য EVE-NG (Emulated Virtual Environment) বর্তমানে একটি ইন্ডাস্ট্রি স্ট্যান্ডার্ড টুল। এটি শুধুমাত্র একটি সিমুলেটর নয়, এটি একটি পূর্ণাঙ্গ ইমুলেটর যেখানে আপনি আসল Router এবং Switch এর অপারেটিং সিস্টেম চালিয়ে রিয়েল-টাইম অভিজ্ঞতা পেতে পারেন।

আজকের এই টিউটোরিয়ালে আমরা শিখবো কীভাবে VMware Workstation-এ ইভ-এনজি ইনস্টল করতে হয়, প্রাথমিক কনফিগারেশন সম্পন্ন করতে হয় এবং সেটআপটি সঠিকভাবে কাজ করছে কি না তা ভেরিফাই করতে হয়।

১. VMware-এ Virtual Machine (VM) তৈরি করা

ইনস্টলেশনের প্রথম ধাপ হলো আপনার পিসির ভেতর একটি ভার্চুয়াল ঘর তৈরি করা যেখানে ইভ-এনজি ওএস থাকবে।

  • Step 1: VMware Workstation ওপেন করুন এবং Create a New Virtual Machine অপশনে ক্লিক করুন।
  • Step 2: ইভ-এনজি থেকে ডাউনলোড করা ISO ফাইলটি সিলেক্ট করুন।
  • Step 3: ওএস টাইপ হিসেবে Linux এবং ভার্সন হিসেবে Ubuntu 64-bit সিলেক্ট করুন।
  • Step 4: অন্তত 8GB RAM এবং 4 CPU Cores বরাদ্দ করুন।

প্রো-টিপ: Virtualization Engine

VMware সেটিংসের Processor ট্যাবে গিয়ে অবশ্যই Virtualize Intel VT-x/EPT or AMD-V/RVI অপশনটি টিক দিয়ে রাখুন। এটি ছাড়া আপনার ইভ-এনজি ল্যাবের ভেতর কোনো নোড (Node) স্টার্ট হবে না।

২. EVE-NG OS ইনস্টলেশন প্রক্রিয়া

ভার্চুয়াল মেশিনটি স্টার্ট করলে ইভ-এনজি ইনস্টলেশন উইজার্ড শুরু হবে।

  • ইনস্টলেশনের সময় আপনাকে ল্যাঙ্গুয়েজ, টাইম জোন এবং কিবোর্ড লেআউট সেট করতে বলা হবে।
  • Partitioning: অটোমেটিক পার্টিশন সিলেক্ট করে এগিয়ে যান।
  • ইনস্টলেশন শেষ হলে ভিএমটি রিবুট হবে এবং আপনি একটি লগইন প্রম্পট দেখতে পাবেন।

৩. প্রাথমিক কনফিগারেশন (Setup Wizard)

প্রথমবার লগইন করার সময় আপনাকে কিছু বেসিক সেটিংস ঠিক করতে হবে।

  • Default Credentials: ইউজারনেম হিসেবে লিখুন root এবং পাসওয়ার্ড হিসেবে eve।
  • লগইন করার পর একটি নীল রঙের স্ক্রিন আসবে।
  • সেখানে আপনাকে নতুন পাসওয়ার্ড সেট করতে হবে (ডিফল্ট eve রাখাই ভালো)।
  • Network Configuration: এখানে DHCP অথবা Static IP সিলেক্ট করুন। ল্যাবের জন্য স্ট্যাটিক আইপি ব্যবহার করা সুবিধাজনক।

৪. ভেরিফিকেশন এবং এক্সেস (Verification)

সেটআপ সঠিকভাবে হয়েছে কি না তা বোঝার জন্য নিচের পদ্ধতিগুলো অনুসরণ করুন:

  • IP Address Check: রাউটারের কনসোলে একটি আইপি অ্যাড্রেস দেখাবে (যেমন: 192.168.x.x)।
  • Web Access: আপনার পিসির যেকোনো ব্রাউজারে গিয়ে ওই আইপিটি টাইপ করুন। যদি ইভ-এনজি লগইন পেজ আসে, তবে আপনার ইনস্টলেশন সফল।
  • Default Web Login: ইউজারনেম: admin এবং পাসওয়ার্ড: eve।

৫. Troubleshooting এবং দরকারি কমান্ড

যদি কোনো সমস্যা হয়, তবে নিচের কমান্ডগুলো আপনার কাজে লাগবে:

১. নেটওয়ার্ক পারমিশন ফিক্স করা:

/opt/unetlab/wrappers/unl_wrapper -a fixpermissions

ব্যাখ্যা: ল্যাবে নতুন ইমেজ আপলোড করার পর এটি চালানো বাধ্যতামূলক। এটি না করলে অনেক সময় ডিভাইস 'Greyed out' হয়ে থাকে।

২. নেটওয়ার্ক স্ট্যাটাস চেক করা:

ifconfig

কমন সমস্যা ও সমাধান:

  • Node Greyed Out: যদি ল্যাবে রাউটার আইকনটি ধূসর বা গ্রে হয়ে থাকে এবং স্টার্ট না হয়, তবে বুঝতে হবে আপনার ইমেজ ঠিকমতো আপলোড হয়নি অথবা Fix Permissions কমান্ডটি চালানো হয়নি।
  • Cannot Access Web UI: নিশ্চিত করুন যে আপনার পিসি এবং ইভ-এনজি ভিএম একই সাবনেটের আওতায় আছে। VMware নেটওয়ার্ক অ্যাডাপ্টারটি Bridge অথবা NAT মোডে আছে কি না চেক করুন।
  • Intel VT-x/AMD-V Disabled: পিসি রিস্টার্ট দিয়ে BIOS সেটিংস থেকে 'Virtualization Technology' এনাবল করুন।

উপসংহার (Conclusion)

EVE-NG ইনস্টল এবং কনফিগার করা একটি কারিগরি দক্ষতা যা আপনার নেটওয়ার্কিং শেখার গতিকে বহুগুণ বাড়িয়ে দেয়। সঠিক ভার্চুয়ালাইজেশন সেটিংস এবং নেটওয়ার্ক কনফিগারেশন নিশ্চিত করলে আপনি কোনো ঝামেলা ছাড়াই আপনার ল্যাব শুরু করতে পারবেন। মনে রাখবেন, একটি সফল ল্যাব সেটআপই হলো একজন দক্ষ নেটওয়ার্ক ইঞ্জিনিয়ার হওয়ার প্রথম ধাপ। পরবর্তী টিউটোরিয়ালে আমরা আলোচনা করবো কীভাবে এই সিস্টেমে বিভিন্ন ভেন্ডরের ইমেজ আপলোড করতে হয়।

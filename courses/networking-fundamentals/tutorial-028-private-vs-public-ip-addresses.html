Public vs Private IP Address

February 27, 2026 Networking Jobaer Hossain

Public vs Private IP Address — Internet এবং Local Network-এর Internal রহস্য উন্মোচন। এই Blog-এ আমরা জানব কেন এই দুই Type-এর IP Address প্রয়োজন এবং তারা কীভাবে একসাথে কাজ করে।

1. Private IP অ্যাড্রেস (Private IP)

একটি Private IP Address হলো আপনার Local Network-এর ভেতরের Device-এর Identity পত্র। একটি বাড়ির ভেতরে Room-গুলোর যেমন Separate নাম থাকে, তেমনি Local Network-এ Device-গুলোর Separate Symbol থাকে এই IP-র মাধ্যমে।

Internal ব্যবহার

এটি শুধুমাত্র Local LAN (LAN) এ কাজ করে। Internet-এর সাথে এর সরাসরি কোনো যোগসূত্র নেই।

অদৃশ্য সুরক্ষা

আপনার Private IP Internet-এর বাইরের কেউ সরাসরি দেখতে পায় না, যা Network-এর প্রাথমিক Security নিশ্চিত করে।

Private IP কি Track করা সম্ভব? (Can We Trace a Private IP?)

একটি Private IP Address শুধুমাত্র যে Local Network-এ এটি Assign করা হয়েছে, তার ভেতরে Track করা সম্ভব। একই Local LAN (LAN)-এর অন্তর্ভুক্ত Device-গুলো এই Private IP দিয়ে যোগাযোগ করতে পারে। তবে, Internet থেকে কোনো Private IP Track বা Access করা সম্ভব নয়। বাইরের Network শুধু Router-এর Public IP-টিই দেখতে পায় NAT (Network Address Translation)-এর কারণ ে।

Private IP-র মূল Feature সমূহ (Key Features)

  • Private IP-গুলো Public Internet থেকে সরাসরি Reach করা যায় না, যা বাইরের Attack থেকে রক্ষা করে।
  • এটি একটি বিশাল Internal Address Space প্রদান করে, যা অনেক Device যুক্ত করার সুযোগ দেয়।
  • প্রতিষ্ঠানগুলো কোনো অতিরিক্ত Public IP না কিনেই তাদের Network-এ বিনামূল্যে Private IP ব্যবহার করতে পারে।
  • বহিরাগত Network-এ যোগাযোগের জন্য NAT বা Proxy Server-এর প্রয়োজন হয়।

সংরক্ষিত Private IP Range-সমূহ

Class A 10.0.0.0 — 10.255.255.255
Class B 172.16.0.0 — 172.31.255.255
Class C 192.168.0.0 — 192.168.255.255

2. Public IP অ্যাড্রেস (Public IP)

Public IP Address হলো সারা পৃথিবীর মানুষের সাথে যোগাযোগের মূল Address। আপনার বাসার একটি রাস্তার Holding Number যেমন Unique থাকে, Internet-এ আপনার Identity ও তেমনি এই Public IP দ্বারা Fixed হয়।

Dynamic IP

Dynamic Public IP সাধারণত আপনার Internet সার্ভিস প্রোভাইডার (ISP) প্রদান করে এবং এটি নির্দিষ্ট সময় পর বা Router Restart করলে পরিবর্তিত হতে পারে। বেশি রভাগ বাসাবাড়ি এবং মোবাইলের Network-এ এটি ব্যবহৃত হয়।

Static IP

Static IP Fixed থাকে এবং কখনোই Change হয় না। এটি সাধারণত Server, Web Hosting এবং Email বা DNS Server-এর জন্য ব্যবহৃত হয়।

Public IP কি Track করা সম্ভব? (Can We Trace a Public IP?)

হ্যাঁ, একটি Public IP Address-কে Track করে Internet-এ সেই Provider বা ISP-কে খুঁজে পাওয়া সম্ভব, এমনকি আনুমানিক ভৌগোলিক অবস্থান (যেমন- শহর বা অঞ্চল) বের করা যায়। তবে এটি দিয়ে কোনো সুনির্দিষ্ট বাড়ির Address বের করা যায় না। গোপনীয়তা-র কারণ ে অনেকেই VPN বা Proxy ব্যবহার করে নিজেদের আসল Public IP আড়াল করে রাখেন।

Public IP-র মূল Feature সমূহ (Key Features)

  • Public IP Address কোনো Device বা Service-কে সরাসরি Internet থেকে Reachable করে তোলে।
  • Website, Application Server, Game Server এবং Cloud Host করার জন্য এটি ব্যবহার করা হয়।
  • NAT বা কোনো Intermediate Routing ছাড়াই Internet-এ সরাসরি যোগাযোগ সম্ভব।
  • Public IP Service Provider-দের কাছ থেকে কিনতে হয়, যা অতিরিক্ত Cost-এর বিষয়।
  • এটি Internet-এ Visible, যার ফলে গোপনীয়তা নিয়ে সচেতন থাকা জরুরি।

3. মূল পার্থক্যসমূহ (Comparison)

Feature Private IP Public IP
যোগাযোগ Local LAN Global Internet
রাউটেবল না (Non-routable) হ্যাঁ (Routable)
অনন্যতা Local Area স্বতন্ত্র বিশ্বব্যাপী স্বতন্ত্র
বরাদ্দানকারী Local Router Internet Provider (ISP)
ব্যয় Free / বিনামূল্যে Paid / মাসিক Fee

4. NAT এর জাদু (How They Work Together)

আপনার Router একটি Bridge (সেতু) হিসেবে কাজ করে।
সে আপনার Device-এর Private IP-কে Internet-এর সামনে Global Public IP-তে রূপান্তর করে।
এই প্রক্রিয়াকেই বলা হয় Network Address Translation (NAT)।

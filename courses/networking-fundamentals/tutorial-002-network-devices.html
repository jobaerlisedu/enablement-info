Network Devices - Hub, Repeater, Bridge, Switch, Router, Gateways and Firewall

07 April, 2026 Networking Jobaer Hossain

Introduction

Network Devices (বা networking hardware) হলো এমন physical appliances যা একটি computer network-এ সংযুক্ত devices-গুলোর মধ্যে communication এবং interaction-এর জন্য প্রয়োজন হয়।

এগুলো ইন্টারনেটের "plumbing"-এর মতো কাজ করে, যা data traffic redirect করে, signals amplify করে এবং connections secure করে। এগুলো কীভাবে কাজ করে তা বোঝার জন্য আমাদেরকে এগুলোকে OSI Model-এর layer অনুযায়ী ভাগ করতে হবে।

Note: এদের functionality-এর উপর ভিত্তি করে এই device-গুলো signal regeneration, frame forwarding, packet switching, routing, protocol conversion এবং access control-এর মতো কাজ করে, যাতে reliable, efficient এবং secure communication নিশ্চিত করা যায়।

Functions of Network Devices

  • Devices-গুলোর মধ্যে Data transmit এবং receive করার মাধ্যমে communication enable করে।
  • Devices-গুলোকে efficiently এবং securely network-এর সাথে connect হতে সাহায্য করে।
  • Congestion কমিয়ে এবং traffic manage করে network performance improve করে।
  • Access control করে এবং unauthorized activities prevent করে security প্রদান করে।
  • Network coverage extend করে এবং signal loss বা attenuation problem solve করে।

Layer 1 Devices: Physical Layer

এই device-গুলো raw electrical বা optical signals (bits) নিয়ে কাজ করে। এগুলো "dumb" device, অর্থাৎ এরা IP Address বা MAC Address বুঝতে পারে না।

1. Hub

  • Function: একটি LAN-এ সংযুক্ত devices-গুলোর জন্য একটি central connection point।
  • How it Works: এটি একটি port-এ আসা incoming signal-কে blindly অন্যান্য সমস্ত port-এ broadcast করে দেয়। এটি জানে না নির্দিষ্ট প্রাপক কে।
  • Drawback: অত্যাধিক network traffic collisions এবং security risks (সবাই সবার data দেখতে পারে)।
  • Status: Obsolete. বর্তমানে Switches দ্বারা replaced হয়ে গেছে।

2. Repeater

  • Function: Network-এর range extend করার জন্য signals regenerate করে।
  • How it Works: এটি একটি weak signal (long cables-এর কারণে হওয়া attenuation-এর জন্য) গ্রহণ করে, এটিকে amplify করে এবং পুনরায় retransmit করে।
  • Use Case: WiFi range বা Ethernet cables-কে 100 meters-এর বাইরে extend করা।

3. Modem (Modulator-Demodulator)

  • Function: আপনার home network-কে ISP (Internet Service Provider)-এর সাথে connect করে।
  • How it Works: আপনার computer-এর digital signals-কে telephone/cable lines-এর জন্য analog signals-এ রূপান্তর করে (Modulation) এবং বিপরীত কাজটাও করে (Demodulation)।

Layer 2 Devices: Data Link Layer

এই device-গুলো MAC Addresses (Physical Addresses) নিয়ে কাজ করে। এগুলো Hub-এর চেয়ে "smarter"।

4. Switch

  • Function: একটি LAN (Local Area Network)-এ devices connect করে।
  • How it Works: Hub-এর মতো না হয়ে, একটি Switch প্রতিটি connected device-এর MAC address শিখে নেয়। এটি data শুধুমাত্র সেই specific port-এ পাঠায় যেখানে destination device সংযুক্ত থাকে।
  • Benefit: Collisions reduce করে এবং security ও speed improve করে।
  • Types:
    • Unmanaged: Plug-and-play (Home/Small Office).
    • Managed: Enterprise ব্যবহারের জন্য Configurable (VLANs, QoS)।

5. Bridge

  • Function: দুটি separate LAN segments-কে এমনভাবে connect করে যেন মনে হয় এটা একটাই network।
  • How it Works: Local traffic-কে local রাখার জন্য MAC addresses-এর উপর ভিত্তি করে traffic filter করে, যা congestion কমায়। বর্তমানে এটি Switches দ্বারা ব্যাপকভাবে replaced হয়ে গেছে (কারণ switch মূলত multi-port bridge ছাড়া আর কিছুই না)।

6. Access Point (AP)

  • Function: একটি Wireless Local Area Network (WLAN) তৈরি করে।
  • How it Works: এটি wired Ethernet এবং wireless WiFi devices-এর মধ্যে একটি bridge হিসেবে কাজ করে।
  • Difference from Router: একটি AP শুধুমাত্র WiFi signal প্রদান করে; এটি traffic route করে না বা IP addresses assign করে না (যদি না এটি একটি "Wireless Router" combo unit হয়)।

Layer 3 Devices: Network Layer

এই device-গুলো IP Addresses (Logical Addresses) নিয়ে কাজ করে। এগুলো বিভিন্ন networks-কে একসাথে connect করে।

7. Router

  • Function: একাধিক networks connect করে (উদাঃ আপনার Home LAN-এর সাথে Internet WAN)।
  • How it Works: Data packets travel করার জন্য best path নির্ধারণ করতে এটি IP Addresses ব্যবহার করে। এই decisions নেয়ার জন্য এটি একটি Routing Table maintain করে।
  • Key Feature: Routers broadcast traffic বন্ধ করে, যা networks-গুলোকে একে অপরের থেকে আলাদা (isolate) রাখে।

8. Brouter (Bridging Router)

  • Function: এটি একটি hybrid device যা Bridge এবং Router উভয়ের মতোই কাজ করে।
  • How it Works: এটি একইসাথে known protocols (যেমন TCP/IP) route করতে পারে এবং unknown protocols (যেমন non-routable legacy traffic) bridge করতে পারে। এটি আজকাল খুব কমই ব্যবহৃত হয়।

Layer 4-7 Devices: Upper Layers

9. Gateway

  • Function: দুটি dissimilar networks-এর মধ্যে একটি translator হিসেবে কাজ করে।
  • How it Works: এটি protocols, data formats বা architectures convert করতে পারে (উদাঃ একটি TCP/IP network-কে একটি legacy Mainframe network-এর সাথে connect করা)।
  • Example: একটি "Internet Gateway" আপনার private LAN requests-গুলোকে public internet requests-এ translate করে।

10. Firewall

  • Function: Security guard। Security rules-এর উপর ভিত্তি করে incoming এবং outgoing traffic control করে।
  • Hardware: Network-এর প্রান্তে (edge) থাকা একটি physical appliance।
  • Software: একটি specific server/PC-তে install করা থাকে।
  • Packet Filtering: Headers (IP/Port) চেক করে।
  • Stateful Inspection: Active connections track করে।

Basics of Computer Networking

31 December, 2025 Networking Jobaer Hossain

Introduction: What is a Computer Network?

Computer Network হলো interconnected devices-এর একটি group, যারা Data এবং resources Share করার জন্য নিজেদের মধ্যে communicate করে। এটি efficient communication-এর সুবিধা দেয় এবং email, file sharing, এবং internet access-এর মতো services সাপোর্ট করে।

  • Nodes হলো সেইসব devices যেমন computers, mobiles, বা printers যারা data exchange করে।
  • Routers এবং switches মূলত information flow কন্ট্রোল এবং direct করে।
  • Transmission media devices-এর মধ্যে data বহন করে।
  • Wired media-র মধ্যে রয়েছে Ethernet এবং optical fiber cables।
  • Wireless media-র মধ্যে রয়েছে Wi-Fi, Bluetooth, এবং infrared।

1. How does a Computer Network work?

একটি shared communication system ব্যবহার করে devices-গুলোর মধ্যে communication এবং data exchange-এর মাধ্যমে Computer Network কাজ করে। Network-এর প্রতিটি device কিছু predefined rules মেনে চলে যাতে accurately, efficiently, এবং securely data transmit হয়।

  • একটি network-এ এমন কিছু nodes থাকে যেমন computers, servers, routers, এবং switches যারা data পাঠায় বা গ্রহণ করে।
  • এই nodes গুলো links-এর মাধ্যমে যুক্ত থাকে, যা wired (cables, optical fiber) বা wireless (Wi-Fi, radio signals) হতে পারে।
  • যখন data পাঠানো হয়, এটি ছোট ছোট packets-এ ভেঙ্গে network-এর মাধ্যমে transmit করা হয়।
  • Protocols বলে দেয় কীভাবে data packets format, transmit, receive, এবং acknowledge করা হবে।
  • প্রতিটি device একটি unique IP address দিয়ে identify করা হয়, যা নিশ্চিত করে data সঠিক গন্তব্যে পৌঁছাচ্ছে কিনা।
  • Switches এবং routers-এর মতো network devices-গুলো available best path দিয়ে data packets forward করে।
  • Firewalls-এর মতো security mechanisms, network traffic monitor করে এবং security rules মেনে data কে পারমিশন কিংবা block করে দেয়।

2. Types of Computer Network Architecture

Computer Network মূলত নিচের broad categories গুলোতে পড়ে:

  • Client-Server Architecture: Client-Server Architecture হলো এমন এক ধরণের Computer Network Architecture যেখানে Nodes গুলো Servers বা Clients হতে পারে। এখানে, server node সাধারণত Client Node Behaviour ম্যানেজ করতে পারে। বিস্তারিত জানতে Client-Server Model পড়ুন।
  • Peer-to-Peer Architecture: P2P (Peer-to-Peer) Architecture-এ Central Server বলে কিছু নেই। প্রতিটি device নিজের ইচ্ছেমতো হয় client বা server হিসেবে কাজ করার স্বাধীনতা রাখে।

3. Essential Network Devices

Router

Router হলো এমন একটি networking device যা multiple networks-কে কানেক্ট করে এবং তাদের মধ্যে data direct করে।

  • Local networks-কে internet-এর সাথে কানেক্ট করে।
  • Data packets-এর জন্য best path বের করে।
  • IP addresses ব্যবহার করে সঠিকভাবে data forward করে।

Switch

Switch মূলত same network-এর মধ্যে থাকা devices গুলোকে কানেক্ট করে এবং internal data communication পরিচালনা করে।

  • Computers, printers, এবং servers কানেক্ট করে।
  • যাকে data পাঠানোর কথা, শুধুমাত্র সেই device-কেই data পাঠায়।
  • Network efficiency এবং performance বাড়ায়।

Hub

Hub একটি basic device যা একটি network-এ multiple devices কানেক্ট করে।

  • কানেক্টেড সব devices-এর কাছেই data broadcast করে।
  • Traffic filter বা manage করে না।
  • Switch-এর তুলনায় কম secure এবং কম efficient।

Bridge

Bridge দুটি network segments-কে কানেক্ট করে এবং তাদের মাঝখানের traffic filter করে।

  • অপ্রয়োজনীয় data transmission কমায়।
  • Network performance ইমপ্রুভ করে।
  • MAC addresses ব্যবহার করে কাজ করে।

Gateway

Gateway ভিন্ন ভিন্ন protocols ব্যবহার করে এমন দুটি আলাদা networks-কে সংযুক্ত করে।

  • ভিন্ন systems-এর মধ্যে data translate করে।
  • Dissimilar networks-এর মধ্যে communication সম্ভব করে।
  • Private networks-কে external networks-এর সাথে যুক্ত করতে বেশি ব্যবহৃত হয়।

Access Point (AP)

Access Point একটি network-এর devices-কে wireless connectivity প্রদান করে।

  • Wired network-কে Wi-Fi-এ extend করে।
  • Mobile devices-কে wirelessly কানেক্ট হতে দেয়।
  • Network coverage area বৃদ্ধি করে।

Modem

Modem ডিজিটাল data-কে transmission-এর উপযোগী signal-এ কনভার্ট করে, এবং ভাইস-ভার্সা।

  • Home বা office network-কে ISP-এর সাথে কানেক্ট করে।
  • Digital signals-কে analog এবং analog-কে digital-এ কনভার্ট করে।
  • Internet access এনেবেল করে।

Firewall

Firewall হলো একটি security device যা network traffic monitor এবং control করে।

  • Unauthorized access ব্লক করে দেয়।
  • Incoming এবং outgoing data filter করে।
  • Cyber threats থেকে networks-কে সুরক্ষিত রাখে।

4. Goals and Uses of Computer Network

  • Communication and Collaboration: Email, messaging, video conferencing, এবং collaborative platforms-এর মাধ্যমে communication সম্ভব করে।
  • Resource Sharing: Multiple users-কে hardware, software, এবং data resources efficiently share করতে দেয়।
  • Data Access and Sharing: Stored data-তে সহজে access দেয় এবং users ও systems-এর মধ্যে information exchange করার সুযোগ দেয়।
  • Internet and Cloud Access: Internet, online services, এবং cloud-based applications-এ access সম্ভব করে তোলে।
  • Cost Efficiency: Shared resources এবং centralized systems ব্যবহারের কারণে operational এবং infrastructure costs কমানো যায়।
  • Reliability and Availability: Backup paths এবং fault-tolerant mechanisms ব্যবহার করে system reliability অনেক বাড়িয়ে তোলে।
  • Scalability and Growth: চাহিদা বাড়ার সাথে সাথে নতুন devices এবং services সহজে যুক্ত করার সাপোর্ট দেয়।
  • Security and Control: Authentication, access control, এবং monitoring ব্যবহার করে data ও network resources প্রটেক্ট করে।

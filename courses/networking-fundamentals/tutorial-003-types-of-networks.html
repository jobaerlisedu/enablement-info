Types of Computer Networks

07 April, 2026 Networking Jobaer Hossain

Computer network হলো একাধিক স্বাধীন computer এবং devices-এর একটি group যা data, resources এবং services share করার জন্য একে অপরের সাথে সংযুক্ত থাকে। এই connection-গুলো wired media অথবা wireless technologies ব্যবহার করে তৈরি করা যায়।

আপনার smartphone থেকে বন্ধুর laptop-এ একটি message পাঠানো, অথবা কোনো remote server-এ store করা files access করার কথা চিন্তা করুন। এই সবকিছুই সম্ভব হয় শুধুমাত্র একটি computer network-এর কারণে।

Classification of Computer Networks

Computer Networks প্রধানত নিচের বিষয়গুলোর উপর ভিত্তি করে classify করা হয়:

  1. Geographical area (ভৌগোলিক বিস্তৃতি)
  2. Transmission medium (ট্রান্সমিশন মিডিয়াম)
  3. Ownership and access control (মালিকানা এবং অ্যাক্সেস কন্ট্রোল)

1. Classification Based on Geographical Area

Geographical area-এর উপর ভিত্তি করে network-গুলোকে তাদের physical distance বা যে area তারা cover করে তার ওপর ভিত্তি করে শ্রেণীবিভাগ করা হয়। Geographical area-এর ওপর ভিত্তি করে Computer networks প্রধানত পাঁচ ধরনের হয়ে থাকে:

i. Personal Area Network (PAN)

একটি Personal Area Network (PAN) একজন user-এর আশেপাশে থাকা personal devices-গুলোকে খুব short range-এর মধ্যে connect করে।

  • Coverage Range: 1–10 meters
  • Key Technologies: Bluetooth, NFC, Infrared
  • Real-World Example: Wireless earbuds-এর সাথে smartphone-এর connection
  • When It Is Used: Personal devices-গুলোর মধ্যে short-range communication-এর জন্য।

ii. Local Area Network (LAN)

একটি Local Area Network (LAN) সাধারণত একটি home, office বা building-এর মতো limited area-এর মধ্যে computers এবং devices-কে connect করে।

  • Coverage Range: একটি রুম, বিল্ডিং, বা ছোট ক্যাম্পাস
  • Key Technologies: Ethernet, Wi-Fi
  • Real-World Example: Home বা office Wi-Fi network
  • When It Is Used: Local location-এর মধ্যে fast এবং secure communication-এর জন্য।

iii. Campus Area Network (CAN)

একটি Campus Area Network (CAN) একটি ক্যাম্পাস বা কাছাকাছি থাকা কয়েকটি বিল্ডিংয়ের একাধিক LAN-কে একত্রে connect করে।

  • Coverage Range: University বা corporate campus
  • Key Technologies: Ethernet, Fiber Optics
  • Real-World Example: College বা corporate campus network
  • When It Is Used: একটি নির্দিষ্ট organization-এর অধীনে থাকা ক্যাম্পাসের একাধিক LAN-কে interconnect করার জন্য।

iv. Metropolitan Area Network (MAN)

একটি Metropolitan Area Network (MAN) কোনো একটি শহর বা metropolitan region-এর মধ্যে থাকা multiple networks-কে connect করে।

  • Coverage Range: শহর বা বড় উপশহর
  • Key Technologies: Fiber Optics, Microwave, Metro Ethernet
  • Real-World Example: City-wide ISP network
  • When It Is Used: পুরো শহর জুড়ে high-speed connectivity দেওয়ার জন্য।

v. Wide Area Network (WAN)

একটি Wide Area Network (WAN) খুব বড় geographical areas যেমন দেশ বা মহাদেশ জুড়ে networks-কে connect করে।

  • Coverage Range: দেশ, মহাদেশ বা গ্লোবাল
  • Key Technologies: Leased lines, Satellite links, Internet
  • Real-World Example: The Internet
  • When It Is Used: Long-distance এবং global communication-এর জন্য।

2. Classification Based on Transmission Technology

Devices-গুলোর মধ্যে কীভাবে data transmit হয় তার ওপর ভিত্তি করেও Computer networks-কে classify করা যায়। Transmission technology বলতে মূলত এক device থেকে অন্য device-এ data signals পাঠানোর method বা medium-কে বোঝায়। এর ওপর ভিত্তি করে network মূলত দুই প্রকার:

i. Broadcast Network

  • Connected থাকা সমস্ত devices-এর মধ্যে একটি shared communication medium ব্যবহার করা হয়।
  • কোনো একটি node থেকে data transmit হলে network-এর বাকি সমস্ত nodes সেটা receive করে।
  • প্রতিটি data packet-এ একটি destination address থাকে, এবং শুধুমাত্র নির্দিষ্ট device-টিই সেই packet-টি process করে।
  • এই communication-টি one-to-many model follow করে।
  • Sender এবং receiver-এর মধ্যে কোনো dedicated physical path থাকে না।
  • বেশিরভাগ ক্ষেত্রেই Local Area Networks (LANs)-এ এটি ব্যবহৃত হয়।
  • ডিজাইন খুবই simple, তবে devices-এর সংখ্যা বাড়লে এর performance কমে যায়।
  • Examples: Ethernet (bus-based), Wireless LAN (Wi-Fi)

ii. Point-to-Point Network

  • Communicating devices-গুলোর মধ্যে dedicated links ব্যবহৃত হয়।
  • একটি node থেকে সরাসরি অন্য একটি node-এ data transmit হয়।
  • এই communication-টি one-to-one model follow করে।
  • Data কখনো কখনো intermediate devices (যেমন routers বা switches) হয়ে যেতে পারে।
  • Best path নির্ধারণ করার জন্য routing algorithms-এর প্রয়োজন হয়।
  • সাধারণত Wide Area Networks (WANs)-এ ব্যবহৃত হয়।
  • Broadcast network-এর তুলনায় এটি অনেক বেশি scalable এবং secure।
  • Examples: Internet, Leased line networks, Telephone networks

3. Classification Based on Ownership and Access Control

Ownership এবং access control বলতে বোঝায় কে একটি network-এর মালিক এবং কাদের এই network ব্যবহার করার permission আছে। এটি মূলত administrative control এবং accessibility-এর উপরে ফোকাস করে। এর ওপর ভিত্তি করে network প্রধানত তিন ধরনের হয়:

i. Private Network

  • কোন একজন ব্যক্তি, organization, বা institution দ্বারা owned এবং managed হয়।
  • শুধুমাত্র authorized users-রাই এটি access করতে পারে।
  • Usernames, passwords, VPNs, বা access tokens-এর মতো authentication-এর প্রয়োজন হয়।
  • মূলত internal communication-এর জন্য ব্যবহৃত হয়।
  • অনেক বেশি security, privacy, এবং control প্রদান করে।
  • Wired বা wireless যেকোনো technology ব্যবহার করে implement করা যায়।
  • Examples: Corporate intranet, Office LAN, Private cloud network

ii. Public Network

  • Service providers বা government authorities দ্বারা owned হয়।
  • General public-এর জন্য accessible।
  • Connect করার জন্য খুব সামান্য অথবা কোনো authorization-এরই প্রয়োজন হয় না।
  • Wide accessibility দিলেও এর security তুলনামূলকভাবে কম।
  • সাধারণত অনেক বিশাল সংখ্যক users একসাথে এটি share করে ব্যবহার করে।
  • Examples: Internet, Public Wi-Fi networks, Mobile cellular networks

iii. Hybrid Network

  • Private এবং public networks-এর একটি combination।
  • Network-এর কিছু অংশ থাকে restricted, আর বাকি অংশ public-ভাবে accessible।
  • Organizations-দেরকে তাদের internal security বজায় রেখে বাইরের বিশ্বের সাথে connectivity রাখার সুবিধা দেয়।
  • Examples: ইন্টারনেটে connected থাকা corporate network, Enterprise cloud networks, Online banking systems

4. Internetwork: Intranet and Extranet

একটি internetwork হলো দুটি বা ততোধিক independent networks-এর সংমিশ্রণ যা একসাথে একটি single logical network হিসেবে কাজ করে। The Internet নিজেই সবচেয়ে বড় internetwork-এর উদাহরণ।

Internetwork Characteristics
শুধুমাত্র individual computers নয় বরং একাধিক networks নিয়ে গঠিত।
Networks-গুলো Size (LAN, MAN, WAN), Technology (wired বা wireless) এবং Architecture-এ ভিন্ন হতে পারে।
Networks-গুলোকে interconnect করার জন্য বিভিন্ন networking devices ব্যবহৃত হয়।
Standard protocols-এর মাধ্যমে communication সংঘটিত হয়।

Internetwork-এর ভেতরে দুটি প্রধান concept রয়েছে: Intranet এবং Extranet।

i. Intranet

একটি intranet হলো সংগঠনের ভেতরে ব্যবহৃত একটি private network। এটা শুধুমাত্র authorized users (যেমন employees) access করতে পারে।

  • Features: একটি সংগঠন দ্বারা owned, login credentials প্রয়োজন, internal communication-এ ব্যবহৃত হয়। TCP/IP, HTTP, FTP ব্যবহার করে।
  • Uses: Employee portals, internal emails, company policies share করা, HR/Payroll applications।
  • Examples: Corporate internal website, University campus portal।

ii. Extranet

একটি extranet হলো intranet-এর একটি extension যা নির্দিষ্ট কিছু external users-কে limited access দেয়। এটা সংগঠনের internal network-কে বিশ্বস্ত বাইরের লোকদের (trusted outsiders) সাথে connect করে।

  • Features: Partially private network, partners/vendors-দের access দেয়, VPN বা secure web access দিয়ে implement করা হয়।
  • Uses: Business-to-business (B2B) communication, Supply chain management, Customer portals।
  • Examples: Online banking portals, Supplier management system।

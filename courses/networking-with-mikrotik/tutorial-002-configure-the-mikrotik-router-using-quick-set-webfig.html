Configure the MikroTik Router using Quick Set & WebFig

28 May, 2026 Networking with MikroTik Jobaer Hossain

MikroTik রাউটার কনফিগার করার কথা ভাবলেই সবার প্রথমে Winbox এর নাম মাথায় আসে। কিন্তু আপনি কি জানেন, কোনো এক্সট্রা সফটওয়্যার ডাউনলোড না করেও শুধু ওয়েব ব্রাউজার ব্যবহার করে একটি MikroTik রাউটার সম্পূর্ণ কনফিগার করা সম্ভব? এর জন্য RouterOS আমাদের দুটি চমৎকার টুল প্রদান করে: Quick Set এবং WebFig।

আজকের এই ডিপ-ডাইভ টিউটোরিয়ালে আমরা শিখবো কীভাবে একটি নতুন MikroTik রাউটারকে বক্স থেকে বের করে ওয়েব ব্রাউজারের মাধ্যমে বেসিক ইন্টারনেট, ওয়াই-ফাই এবং লোকাল নেটওয়ার্ক সেটআপ করা যায়।

Quick Set এবং WebFig কী?

WebFig: এটি হলো RouterOS এর একটি ওয়েব-বেসড গ্রাফিক্যাল ইউজার ইন্টারফেস (GUI)। Winbox-এ আপনি যেসব মেনু বা অপশন পান, WebFig-এ ঠিক একই অপশনগুলো ওয়েব ব্রাউজারে পাওয়া যায়।

Quick Set: এটি WebFig (বা Winbox) এর ভেতরের একটি বিশেষ ড্যাশবোর্ড, যেখানে রাউটারের সবচেয়ে সাধারণ সেটিংসগুলো (যেমন WAN, LAN, Wi-Fi, এবং Password) একটিমাত্র স্ক্রিনে গুছিয়ে দেওয়া থাকে। মাত্র কয়েক ক্লিকে রাউটার রেডি করার জন্য এটি ডিজাইন করা হয়েছে।

রিয়েল-লাইফ সিনারিও: আপনি একজন ক্লায়েন্টের অফিসে গেছেন রাউটার কনফিগার করতে, কিন্তু আপনার ল্যাপটপে Winbox নেই বা ক্লায়েন্ট তার পিসিতে থার্ড-পার্টি সফটওয়্যার ইন্সটল করতে দিচ্ছে না। তখন আপনি ব্রাউজার ওপেন করে WebFig এবং Quick Set এর মাধ্যমে কয়েক মিনিটেই রাউটারটি লাইভ করে দিতে পারবেন।

১. Router-এ লগইন করার প্রাথমিক ধাপ

একটি নতুন বা ফ্যাক্টরি রিসেট করা MikroTik রাউটারে লগইন করার জন্য নিচের ধাপগুলো অনুসরণ করুন:

  • কানেকশন: রাউটারের Ether1 পোর্টে ইন্টারনেটের তার (ISP এর ক্যাবল) লাগান। এরপর রাউটারের Ether2 (বা অন্য যেকোনো ল্যান পোর্ট) থেকে একটি ক্যাবল নিয়ে আপনার পিসিতে লাগান।
  • IP Address: ডিফল্টভাবে MikroTik রাউটারের পিসির দিকের পোর্টগুলোতে একটি DHCP Server চালু থাকে এবং রাউটারের ডিফল্ট আইপি হয় 192.168.88.1। আপনার পিসি স্বয়ংক্রিয়ভাবে 192.168.88.X ব্লকের একটি আইপি পেয়ে যাবে।
  • ব্রাউজারে প্রবেশ: আপনার পিসির যেকোনো ওয়েব ব্রাউজার (যেমন Chrome, Edge) ওপেন করে অ্যাড্রেস বারে http://192.168.88.1 লিখে এন্টার চাপুন।
  • লগইন: লগইন পেজ আসলে Username এ admin লিখুন এবং Password এর ঘর খালি (Blank) রেখে লগইন বাটনে ক্লিক করুন।

২. Quick Set ব্যবহার করে বেসিক কনফিগারেশন

লগইন করার পর ডিফল্টভাবেই Quick Set পেজটি ওপেন হয়। এটি অত্যন্ত ইউজার ফ্রেন্ডলি। চলুন ধাপে ধাপে এটি কনফিগার করি:

Step 1: Mode নির্বাচন করা

পেজের একদম উপরে বাম কোণায় একটি ড্রপডাউন মেনু থাকে। হোম বা ছোট অফিসের জন্য এখান থেকে Home AP Dual বা Ethernet (Router) সিলেক্ট করুন। ওয়্যারলেস রাউটার হলে Home AP সিলেক্ট করাই বেস্ট।

Step 2: Internet (WAN) কনফিগারেশন

এই সেকশনে আপনার ISP (ইন্টারনেট সার্ভিস প্রোভাইডার) এর দেওয়া কানেকশন টাইপ সেট করতে হবে। এখানে তিনটি অপশন থাকে:

  • Static: আপনার ISP যদি আপনাকে ফিক্সড IP, Netmask, Gateway এবং DNS দেয়, তবে সেগুলো এখানে বসান।
  • PPPoE: আপনার ISP যদি আপনাকে Username এবং Password দেয়, তবে PPPoE সিলেক্ট করে সেগুলো বসিয়ে দিন।
  • Automatic (DHCP): আইএসপি যদি অটোমেটিক আইপি দেয়, তবে এটি সিলেক্ট করুন।

নোট: ম্যাক ফিল্টারিং থাকলে নিচে থাকা 'MAC Address' এর ঘরে আপনার পুরনো রাউটারের ম্যাক বসিয়ে ক্লোন (Clone) করে নিতে পারবেন।

Step 3: Local Network (LAN) কনফিগারেশন

এই অংশে আপনি নির্ধারণ করবেন আপনার পিসি বা ল্যাপটপগুলো কোন ব্লকের IP Address পাবে।

  • IP Address: ডিফল্ট 192.168.88.1 এর বদলে আপনি চাইলে 192.168.1.1 বা আপনার পছন্দমতো আইপি বসাতে পারেন। Netmask সাধারণত 255.255.255.0 (/24) থাকে।
  • DHCP Server: এটি অবশ্যই চেকমার্ক করে রাখবেন এবং IP Range (যেমন 192.168.1.10-192.168.1.254) ঠিক করে দেবেন, যাতে পিসিগুলো অটোমেটিক আইপি পায়।
  • NAT: এই অপশনটি অবশ্যই চেকমার্ক করে রাখবেন। NAT (Network Address Translation) না থাকলে আপনার লোকাল ডিভাইসগুলো ইন্টারনেটে যেতে পারবে না।

Step 4: Wireless (Wi-Fi) কনফিগারেশন

আপনার রাউটারে যদি Wi-Fi থাকে, তবে পেজের ডান দিকে Wireless সেকশনটি পাবেন।

  • Network Name (SSID): এখানে আপনার ওয়াই-ফাইয়ের নাম দিন (যেমন: MyOffice_WiFi)।
  • WiFi Password: পাসওয়ার্ডের ঘরে একটি শক্তিশালী পাসওয়ার্ড দিন।
  • Country: দেশের জায়গায় Bangladesh সিলেক্ট করতে পারেন। এটি ফ্রিকোয়েন্সি রেগুলেশন মেনে কাজ করতে সাহায্য করে।

Step 5: System Configuration

সবশেষে, পেজের নিচের দিকে Router Identity এর ঘরে রাউটারের একটি নাম দিন এবং Password এর ঘরে রাউটারে লগইন করার জন্য একটি নতুন অ্যাডমিন পাসওয়ার্ড সেট করে Apply Configuration বাটনে ক্লিক করুন। আপনার বেসিক রাউটার সেটআপ সম্পন্ন!

৩. WebFig ব্যবহার করে অ্যাডভান্সড কনফিগারেশন

Quick Set এর কাজ শেষ হওয়ার পর আপনি যদি আরও অ্যাডভান্সড কোনো কাজ (যেমন Bandwidth Control, Port Forwarding) করতে চান, তবে আপনাকে WebFig মেনুগুলো ব্যবহার করতে হবে।

লগইন করা অবস্থায় পেজের ডানদিকের একদম ওপরে WebFig বাটনে ক্লিক করুন। এখন দেখবেন বাম পাশে Winbox এর মতো বিশাল একটি মেনু লিস্ট চলে এসেছে।

WebFig মেনু কাজের ধরন (Functionality)
IP > Addresses এখান থেকে আপনি রাউটারের যেকোনো ফিজিক্যাল বা লজিক্যাল পোর্টে (VLAN) নতুন IP Address বসাতে পারবেন।
IP > Routes রাউটিং টেবিল চেক করা এবং নতুন Static Route বা Default Gateway (0.0.0.0/0) অ্যাড করা।
IP > Firewall এটি সবচেয়ে গুরুত্বপূর্ণ অংশ। এখান থেকে Filter Rules, NAT (Masquerade), এবং Mangle রুলস তৈরি করা যায়।
Queues কোন ইউজার কতটুকু স্পিড (Bandwidth) পাবে, তা Simple Queues এর মাধ্যমে এখান থেকে লিমিট করা হয়।

৪. কমান্ড লাইন ইন্টারফেস (CLI) - বিকল্প পদ্ধতি

যারা ওয়েব ইন্টারফেসের পাশাপাশি কমান্ড লাইনে কাজ করতে পছন্দ করেন, তারা WebFig এর উপরের ডানদিকের Terminal বাটনে ক্লিক করে কমান্ড প্রম্পট ওপেন করতে পারেন। Quick Set-এ আমরা যা যা করেছি, তার বেসিক কমান্ডগুলো হলো:

ল্যান (LAN) আইপি বসানো:

/ip address add address=192.168.1.1/24 interface=bridge

NAT (Masquerade) কনফিগার করা:

/ip firewall nat add chain=srcnat out-interface=ether1 action=masquerade

ডিফল্ট রাউট (Internet Gateway) অ্যাড করা:

/ip route add dst-address=0.0.0.0/0 gateway=ISP_Gateway_IP

৫. Troubleshooting Tips (সাধারণ সমস্যা ও সমাধান)

WebFig বা Quick Set ব্যবহার করার সময় কিছু সাধারণ সমস্যা হতে পারে। নিচে সেগুলোর সমাধান দেওয়া হলো:

  • WebFig পেজ ওপেন হচ্ছে না: আপনার পিসির আইপি অ্যাড্রেস চেক করুন। পিসিতে যদি ম্যানুয়ালি অন্য কোনো আইপি বসানো থাকে, তবে তা রিমুভ করে Obtain an IP address automatically সিলেক্ট করুন। পিসি যেন 192.168.88.x ব্লকের আইপি পায় তা নিশ্চিত করুন।
  • রাউটার ইন্টারনেট পাচ্ছে, কিন্তু পিসি পাচ্ছে না: WebFig থেকে IP > Firewall > NAT এ যান। চেক করুন srcnat চেইনে action=masquerade নামের কোনো রুল আছে কি না। না থাকলে এটি তৈরি করুন।
  • IP ভুলে গেলে (MAC Login): ওয়েব ব্রাউজার শুধু IP Address দিয়ে কাজ করে। আপনি যদি রাউটারের আইপি ভুলে যান বা কনফিগারেশন ভুল করে ফেলেন, তবে ব্রাউজার দিয়ে আর লগইন করা যাবে না। তখন আপনাকে বাধ্য হয়ে Winbox ডাউনলোড করে MAC Address এর মাধ্যমে লগইন করতে হবে।

উপসংহার (Conclusion)

MikroTik কনফিগারেশন মানেই যে শুধু Winbox বা কঠিন CLI কমান্ড, এই ধারণাটি ভুল। Quick Set এবং WebFig এর মাধ্যমে যেকোনো নতুন বা সাধারণ ব্যবহারকারী মাত্র কয়েকটি ক্লিকেই একটি প্রফেশনাল রাউটার সেটআপ করে ফেলতে পারেন। Quick Set আপনাকে দেয় দ্রুততার নিশ্চয়তা, আর WebFig আপনাকে দেয় রাউটারের অ্যাডভান্সড ফিচারগুলো ব্রাউজার থেকেই নিয়ন্ত্রণ করার স্বাধীনতা। এই দুটি টুলের সঠিক ব্যবহার জানা থাকলে যেকোনো পরিস্থিতিতে আপনি একটি MikroTik নেটওয়ার্ক সহজে ম্যানেজ করতে পারবেন।

Installing and Configuring The Dude Server and Client

24 April, 2026 Networking with MikroTik Jobaer Hossain

একটি ছোট নেটওয়ার্ক ম্যানেজ করা সহজ, কিন্তু যখন আপনার নেটওয়ার্কে ডজন ডজন Router, Switch, এবং Access Point থাকে, তখন ম্যানুয়ালি সব চেক করা অসম্ভব হয়ে পড়ে। কোন ডিভাইসটি ডাউন হলো, কোন পোর্টে ট্রাফিক বেশি—এই সব রিয়েল-টাইমে দেখার জন্য আমাদের প্রয়োজন একটি Network Management System (NMS)।

MikroTik-এর নিজস্ব ফ্রি মনিটরিং টুল হলো The Dude। এটি অত্যন্ত শক্তিশালী একটি অ্যাপ্লিকেশন যা স্বয়ংক্রিয়ভাবে আপনার নেটওয়ার্কের ম্যাপ তৈরি করতে পারে এবং ডিভাইসের সমস্যা হলে আপনাকে অ্যালার্ট পাঠাতে পারে। আজকের এই ডিপ-ডাইভ টিউটোরিয়ালে আমরা শিখবো কীভাবে The Dude Server এবং Client ইনস্টল ও কনফিগার করতে হয়।

The Dude কী এবং কেন ব্যবহার করবেন?

The Dude মূলত দুটি অংশে বিভক্ত: Server (যা ডিভাইসের ডাটা সংগ্রহ করে এবং রাউটারে চলে) এবং Client (যা দিয়ে আপনি কম্পিউটার থেকে ম্যাপ এবং ডাটা দেখেন)।

  • Network Discovery: এটি স্বয়ংক্রিয়ভাবে নেটওয়ার্ক স্ক্যান করে নতুন ডিভাইস খুঁজে পায়।
  • Monitoring: ICMP, DNS, HTTP বা SNMP প্রোটোকল ব্যবহার করে ডিভাইসের সার্ভিস মনিটর করে।
  • Visual Mapping: পুরো নেটওয়ার্কের একটি গ্রাফিক্যাল ম্যাপ তৈরি করে দেয়।
  • Alerting: কোনো ডিভাইস ডাউন হলে ইমেইল বা বিপ সাউন্ডের মাধ্যমে আপনাকে অবগত করে।

১. The Dude Server ইনস্টলেশন (MikroTik RouterOS)

The Dude Server বর্তমানে শুধুমাত্র ARM, ARM64, MMIPS এবং x86 আর্কিটেকচারের রাউটারে চলে (যেমন: RB3011, RB4011, CCR সিরিজ)। এটি ইনস্টল করার জন্য আপনার রাউটারের নির্দিষ্ট .npk ফাইল প্রয়োজন।

ধাপ ১: সঠিক ফাইল ডাউনলোড করা

  • প্রথমে MikroTik-এর অফিশিয়াল Download Page-এ যান।
  • আপনার রাউটারের আর্কিটেকচার অনুযায়ী Extra packages জিপ ফাইলটি ডাউনলোড করুন।
  • জিপ ফাইলটি আনজিপ (Unzip) করলে আপনি dude-x.x.npk ফাইলটি পাবেন।

ধাপ ২: রাউটারে ফাইল আপলোড করা

  • Winbox দিয়ে রাউটারে লগইন করুন।
  • বাম পাশের মেনু থেকে Files ওপেন করুন।
  • আপনার পিসি থেকে ডাউনলোড করা dude.npk ফাইলটি ড্র্যাগ করে Winbox-এর File List-এ ছেড়ে দিন।

ধাপ ৩: রাউটার রিবুট এবং অ্যাক্টিভেশন

  • ফাইল আপলোড হয়ে গেলে System > Reboot-এ গিয়ে রাউটারটি রিবুট দিন।
  • রিবুট হওয়ার পর Winbox মেনুতে Dude নামের একটি নতুন অপশন দেখতে পাবেন।
  • টার্মিনাল থেকে নিচের কমান্ডটি দিয়ে সার্ভারটি এনাবল করুন:
/dude set enabled=yes

২. The Dude Client ইনস্টলেশন (Windows PC)

সার্ভার রেডি হয়ে গেলে এখন আমাদের কম্পিউটারে ক্লায়েন্ট সফটওয়্যারটি নামাতে হবে যাতে আমরা ম্যাপটি দেখতে পারি।

ইনস্টলেশন ধাপসমূহ:

  • MikroTik ডাউনলোড পেইজ থেকে The Dude Client এক্সিকিউটেবল (.exe) ফাইলটি ডাউনলোড করুন।
  • সাধারণ সফটওয়্যারের মতোই এটি উইন্ডোজে ইনস্টল করুন।
  • ইনস্টল শেষে ওপেন করলে একটি লগইন উইন্ডো আসবে।
  • Server: এর ঘরে আপনার রাউটারের IP দিন।
  • Username এবং Password হিসেবে আপনার রাউটারের লগইন ডিটেইলস দিন।
  • Connect বাটনে ক্লিক করুন।

৩. নেটওয়ার্ক ডিসকভারি এবং ম্যাপ তৈরি করা

প্রথমবার কানেক্ট হওয়ার পর The Dude আপনাকে একটি ডিসকভারি রান করতে বলবে।

  • Discover বাটনে ক্লিক করুন।
  • Subnet: এর ঘরে আপনার লোকাল নেটওয়ার্ক রেঞ্জ দিন (যেমন: 192.168.88.0/24)।
  • The Dude তখন নেটওয়ার্ক স্ক্যান শুরু করবে এবং আপনার রাউটার, পিসি ও সুইচগুলোকে ম্যাপে আইকন আকারে সাজাবে।
  • ডিভাইসগুলো সবুজ হলে বুঝবেন সেগুলো Up আছে, লাল হলে Down।

৪. অ্যাডভান্সড কনফিগারেশন: SNMP মনিটরিং

সাধারণ পিং ছাড়াও ডিভাইসের CPU লোড বা পোর্টের লাইভ স্পিড দেখার জন্য SNMP (Simple Network Management Protocol) কনফিগার করা জরুরি।

MikroTik রাউটারে SNMP এনাবল করার কমান্ড:

/snmp set enabled=yes contact="Jobaer Hossain" location="Main Office"
/snmp community set [find name=public] addresses=0.0.0.0/0

এরপর The Dude ক্লায়েন্টে ওই ডিভাইসের ওপর রাইট-ক্লিক করে Settings-এ যান এবং SNMP ট্যাব থেকে public কমিউনিটি সিলেক্ট করে দিন। এখন আপনি ডিভাইসের ওপর মাউস রাখলেই লাইভ গ্রাফ দেখতে পাবেন।

৫. Troubleshooting Tips (সাধারণ সমস্যা ও সমাধান)

সমস্যা (Problem) সমাধান (Solution)
Connection Refused নিশ্চিত করুন যে রাউটারে Dude Server এনাবল করা আছে (/dude set enabled=yes)। রাউটারে কোনো ফায়ারওয়াল রুল Dude-এর পোর্ট (Port 8291 বা 2211) ব্লক করছে কি না চেক করুন।
Data Storage issue The Dude অনেক ডাটা জেনারেট করে। আপনার রাউটারের ইন্টারনাল স্টোরেজ কম হলে System > Disk-এ গিয়ে একটি এক্সটারনাল SSD বা USB ড্রাইভ মাউন্ট করুন এবং Dude সেটিংসে ডাটাবেজ পাথ সেখানে সরিয়ে দিন।
Version Mismatch আপনার RouterOS ভার্সন এবং The Dude Client ভার্সন অবশ্যই এক হতে হবে। রাউটার আপডেট করলে ক্লায়েন্ট সফটওয়্যারটিও আপডেট করুন।

প্রফেশনাল টিপস:

  • Custom Icons: ম্যাপকে সুন্দর করতে রাউটার, রেডিও লিংকের জন্য কাস্টম ইমেজ আপলোড করুন।
  • Dependencies: ধরুন আপনার একটি কোর সুইচ ডাউন হয়েছে। তার পেছনে থাকা ১০টি রাউটারও ডাউন দেখাবে। Dependency সেট করলে The Dude শুধু কোর সুইচটিকেই লাল দেখাবে, বাকিগুলোকে "Unreachable" মার্ক করবে। এতে মূল সমস্যা খুঁজে পাওয়া সহজ হয়।

উপসংহার (Conclusion)

The Dude হলো এমন একটি টুল যা একজন নেটওয়ার্ক অ্যাডমিনিস্ট্রেটরের জীবনকে সহজ করে তোলে। শুধুমাত্র ডিভাইসের আইপি মনে রাখার বদলে একটি ভিজ্যুয়াল ম্যাপের মাধ্যমে পুরো নেটওয়ার্কের হেলথ মনিটর করা অনেক বেশি প্রফেশনাল। সার্ভার এবং ক্লায়েন্টের সঠিক কনফিগারেশনের মাধ্যমে আপনি আপনার নেটওয়ার্ককে অনেক বেশি স্টেবল এবং প্রডাক্টিভ করতে পারবেন। মনে রাখবেন, একটি মনিটর করা নেটওয়ার্ক সব সময় একটি আন-মনিটর করা নেটওয়ার্কের চেয়ে দ্রুত রিকভার হয়।

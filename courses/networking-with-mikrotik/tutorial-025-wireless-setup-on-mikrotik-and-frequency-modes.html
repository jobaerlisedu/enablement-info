Wireless setup on MikroTik and frequency modes

10 October, 2026 Networking with MikroTik Jobaer Hossain

MikroTik রাউটার দিয়ে Wireless Network তৈরি করা অন্যান্য সাধারণ হোম রাউটারের (যেমন TP-Link বা Tenda) মতো এক ক্লিকের কাজ নয়। এখানে আপনাকে সিকিউরিটি, ট্রান্সমিট পাওয়ার (Tx Power), এবং সবচেয়ে গুরুত্বপূর্ণ Frequency Mode নিয়ে সূক্ষ্ম কনফিগারেশন করতে হয়। একটি ভুল কনফিগারেশনের কারণে আপনার নেটওয়ার্কে প্রচুর নয়েজ (Noise) তৈরি হতে পারে, আবার আইনের দিক থেকেও আপনি বিপদে পড়তে পারেন।

আজকের এই ডিপ-ডাইভ টিউটোরিয়ালে আমরা শিখবো কীভাবে একটি MikroTik রাউটারকে Access Point হিসেবে কনফিগার করতে হয়, Security Profile কীভাবে তৈরি করতে হয়, এবং এর Frequency Modes (যেমন: regulatory-domain, manual-txpower, superchannel) এর পেছনের কনসেপ্টগুলো কী।

রিয়েল-লাইফ সিনারিও (The Setup)

ধরি, আপনি আপনার অফিসে একটি MikroTik hAP ac2 রাউটার বসিয়েছেন। আপনি চান আপনার অফিসের এমপ্লয়িরা Office_WiFi নামের একটি সিগন্যালে কানেক্ট করে ইন্টারনেট ব্যবহার করুক। এর জন্য আমাদের রাউটারের Wireless ইন্টারফেসটি (wlan1) চালু করতে হবে এবং এটিকে AP Bridge মোডে কনফিগার করতে হবে। কিন্তু তার আগে আমাদের পাসওয়ার্ডের একটি প্রোফাইল তৈরি করে নিতে হবে।

১. Security Profile তৈরি করা (পাসওয়ার্ড সেটআপ)

Wireless ইন্টারফেস চালু করার আগে একটি সিকিউরিটি প্রোফাইল তৈরি করা বেস্ট প্র্যাকটিস। এটি না করলে আপনার ওয়াই-ফাই নেটওয়ার্কটি ওপেন (Open) বা পাসওয়ার্ড ছাড়া থেকে যাবে।

  • Winbox থেকে বাম পাশের মেনু Wireless এ ক্লিক করুন।
  • Security Profiles ট্যাবে যান এবং + (Add) আইকনে ক্লিক করুন।
  • Name: এর ঘরে একটি নাম দিন (যেমন: office-secure)।
  • Mode: এর ঘরে dynamic keys সিলেক্ট করুন।
  • Authentication Types: এর ঘরে WPA2 PSK চেক বক্সে টিক দিন (এটি বর্তমানে সবচেয়ে স্ট্যান্ডার্ড)।
  • WPA2 Pre-Shared Key: এর ঘরে আপনার কাঙ্ক্ষিত ওয়াই-ফাই পাসওয়ার্ডটি দিন (যেমন: SecretPass2026)।
  • Apply এবং OK দিন।

২. Wireless Interface কনফিগারেশন (AP Setup)

এখন আমরা মূল ওয়্যারলেস ইন্টারফেসটি কনফিগার করবো এবং সেটিতে সিকিউরিটি প্রোফাইলটি জুড়ে দেবো।

  • Wireless মেনুর WiFi Interfaces ট্যাবে যান।
  • ডিফল্টভাবে wlan1 ইন্টারফেসটি ধূসর (Gray/Disabled) থাকে। সেটিতে ক্লিক করে উপরের ✓ (Enable) আইকনে ক্লিক করে চালু করুন।
  • এবার wlan1 এর ওপর ডাবল-ক্লিক করে Wireless ট্যাবে যান।
  • Mode: ap bridge সিলেক্ট করুন (যাতে একাধিক ডিভাইস কানেক্ট হতে পারে)।
  • Band: 2GHz-B/G/N সিলেক্ট করুন (সাধারণ রাউটারের জন্য)।
  • Channel Width: 20/40MHz Ce দিন (ভালো স্পিডের জন্য)।
  • SSID: Office_WiFi দিন (এটি আপনার ওয়াই-ফাইয়ের নাম)।
  • Security Profile: ড্রপডাউন থেকে একটু আগে তৈরি করা office-secure সিলেক্ট করুন।

৩. Frequency Modes: ডিপ-ডাইভ এক্সপ্ল্যানেশন

MikroTik ওয়্যারলেস কনফিগারেশনের সবচেয়ে অ্যাডভান্সড এবং কনফিউজিং অংশ হলো Frequency Mode। Wireless ট্যাবের Advanced Mode বাটনে ক্লিক করলে আপনি এই অপশনটি দেখতে পাবেন। এখানে প্রধানত তিনটি মোড থাকে:

Frequency Mode কাজের বিবরণ এবং প্রভাব (Behavior & Impact) ব্যবহারের ক্ষেত্র (Use Case)
regulatory-domain এটি সবচেয়ে স্ট্যান্ডার্ড এবং লিগ্যাল (Legal) মোড। এটি সিলেক্ট করার পর আপনাকে Country (দেশ) সিলেক্ট করতে হয়। রাউটার তখন ওই দেশের টেলিযোগাযোগ আইন (যেমন BTRC) অনুযায়ী নির্দিষ্ট ফ্রিকোয়েন্সি এবং ম্যাক্সিমাম Tx Power এর লিমিট নিজে থেকেই সেট করে নেয়। হোম, অফিস বা যেকোনো রেগুলার Access Point সেটআপের জন্য এটি রিকমেন্ডেড। এটি আপনার ডিভাইসকে আইনি ঝামেলার বাইরে রাখে।
manual-txpower এই মোডে রাউটার আপনার সিলেক্ট করা দেশের ফ্রিকোয়েন্সি লিস্টই ফলো করে, কিন্তু Tx Power (অ্যান্টেনার শক্তি) এর ওপর দেশের আইন আর কাজ করে না। আপনি চাইলে ম্যানুয়ালি ট্রান্সমিট পাওয়ার বাড়াতে বা কমাতে পারবেন। যাদের নিজস্ব ল্যাবে টেস্টিং করার জন্য নির্দিষ্ট ফ্রিকোয়েন্সির ওপর বেশি শক্তি (Tx Power) প্রয়োগ করার প্রয়োজন হয়।
superchannel এটি হলো MikroTik এর একটি "হ্যাক" বা আনলকড মোড। এটি রাউটারের হার্ডওয়্যার যে কয়টি ফ্রিকোয়েন্সি সাপোর্ট করে, তার সব কটি আনলক করে দেয়—দেশের আইন যাই হোক না কেন। এতে অনেক কাস্টম বা লুকানো ফ্রিকোয়েন্সি পাওয়া যায়। Point-to-Point (PTP) লিংকের ক্ষেত্রে যখন রেগুলার ফ্রিকোয়েন্সিগুলোতে প্রচুর নয়েজ (Noise) থাকে, তখন আইএসপিরা ফাঁকা এবং আনকমন ফ্রিকোয়েন্সি ব্যবহার করার জন্য এটি ব্যবহার করে। (নোট: লাইসেন্স ছাড়া এটি ব্যবহার করা বেআইনি হতে পারে।)

Country Code সেট করা (Regulatory Domain এর ক্ষেত্রে):

যদি আপনি Frequency Mode হিসেবে regulatory-domain সিলেক্ট করেন, তবে Country: ফিল্ডে Bangladesh (বা আপনার দেশ) সিলেক্ট করা বাধ্যতামূলক। এর ফলে রাউটার বুঝতে পারবে সে 2.4GHz এ 1 থেকে 13 চ্যানেল পর্যন্ত ব্যবহার করতে পারবে এবং DFS (Dynamic Frequency Selection) রুল মেনে চলবে।

৪. Command Line (CLI) ব্যবহার করে Wireless Setup

প্রফেশনালরা টার্মিনাল ব্যবহার করে খুব দ্রুত এই কনফিগারেশনগুলো করতে পারেন।

Security Profile তৈরি করা:

[admin@MikroTik] > /interface wireless security-profiles add name=office-secure mode=dynamic-keys authentication-types=wpa2-psk wpa2-pre-shared-key="SecretPass2026"

Wireless Interface (AP) এবং Frequency Mode কনফিগার করা:

[admin@MikroTik] > /interface wireless set wlan1 disabled=no mode=ap-bridge band=2ghz-b/g/n channel-width=20/40mhz-Ce ssid="Office_WiFi" security-profile=office-secure frequency-mode=regulatory-domain country=bangladesh

৫. Troubleshooting Tips (সাধারণ সমস্যা ও সমাধান)

  • মোবাইল বা ল্যাপটপ কানেক্ট হচ্ছে না (Authentication Timeout): এটি সাধারণত ভুল পাসওয়ার্ড দেওয়ার কারণে হয়। যদি আপনি নিশ্চিত থাকেন পাসওয়ার্ড ঠিক আছে, তবে Security Profile এ গিয়ে WPA (TKIP) এর টিক তুলে শুধু WPA2 (AES) রাখুন। আধুনিক ডিভাইসগুলো TKIP সাপোর্ট করে না।
  • Country Code সিলেক্ট করার পর স্পিড কমে যাওয়া: regulatory-domain সেট করলে অনেক দেশে Tx Power এর লিমিট 20 dBm (100mW) এ নেমে আসে। এর ফলে কভারেজ কমে যেতে পারে। যদি কভারেজ বাড়াতে হয়, তবে Frequency Mode পরিবর্তন করে manual-txpower দিয়ে দেখতে পারেন (তবে এটি নিজ দায়িত্বে করতে হবে)।
  • Superchannel মোডে ক্লায়েন্ট কানেক্ট না হওয়া: আপনি যদি PTP লিংকে Access Point কে superchannel করে কোনো আনকমন ফ্রিকোয়েন্সি (যেমন 2312 MHz) দেন, তবে অপর প্রান্তের (Station) রাউটারকেও superchannel মোডে থাকতে হবে। নতুবা ক্লায়েন্ট রাউটার ওই ফ্রিকোয়েন্সি স্ক্যান করে খুঁজে পাবে না।

উপসংহার (Conclusion)

MikroTik Wireless Setup এর ক্ষেত্রে Security Profile এবং Frequency Mode এর সঠিক কনফিগারেশন অত্যন্ত তাৎপর্যপূর্ণ। সাধারণ অফিস বা হোম সেটআপের জন্য সব সময় regulatory-domain ব্যবহার করা বেস্ট প্র্যাকটিস। কিন্তু যদি আপনি লং ডিসট্যান্স PTP Link বা ISP নেটওয়ার্কে কাজ করেন, তবে superchannel ব্যবহার করে নয়েজ-ফ্রি ফ্রিকোয়েন্সি খুঁজে বের করাটা আপনার জন্য একটি "লাইফ সেভার" হিসেবে কাজ করতে পারে। প্রতিটি সেটিংয়ের ইমপ্যাক্ট (Impact) বুঝে কনফিগারেশন করলে আপনার ওয়্যারলেস নেটওয়ার্ক হবে সুপার-ফাস্ট এবং স্টেবল।

Sending emails from the MikroTik router

15 June, 2026 Automation with MikroTik Jobaer Hossain

একজন নেটওয়ার্ক অ্যাডমিনিস্ট্রেটর হিসেবে ২৪ ঘণ্টা রাউটারের দিকে তাকিয়ে বসে থাকা সম্ভব নয়। রাউটারের কোনো নির্দিষ্ট লিংকে সমস্যা হলে, ইন্টারনেট ডিসকানেক্ট হলে, কিংবা কোনো হ্যাকিং প্রচেষ্টা (Brute-force Attack) হলে আপনার কাছে যদি স্বয়ংক্রিয়ভাবে একটি অ্যালার্ট চলে আসে, তবে কেমন হয়? অথবা, প্রতিদিন রাত ১২টায় রাউটার নিজে নিজেই তার ব্যাকআপ (Backup) ফাইলটি আপনার ইমেইলে পাঠিয়ে দেয়?

MikroTik RouterOS-এর বিল্ট-ইন Email Tool ব্যবহার করে এই কাজগুলো অত্যন্ত নিখুঁতভাবে করা সম্ভব। আজকের এই ডিপ-ডাইভ টিউটোরিয়ালে আমরা শিখবো কীভাবে Gmail-এর SMTP Server ব্যবহার করে আপনার রাউটারকে স্বয়ংক্রিয় ইমেইল পাঠানোর উপযোগী করে তুলতে হয় এবং একটি রিয়েল-লাইফ ব্যাকআপ স্ক্রিপ্ট কনফিগার করতে হয়।

রিয়েল-লাইফ সিনারিও (The Automated Backup)

সমস্যা: রাউটারের কনফিগারেশন অনেক জটিল। কোনো কারণে রাউটারের হার্ডওয়্যার নষ্ট হয়ে গেলে বা রিসেট হয়ে গেলে পুরো কনফিগারেশন নতুন করে করা অত্যন্ত সময়সাপেক্ষ। ম্যানুয়ালি প্রতিদিন ব্যাকআপ নেওয়াটাও বিরক্তির কারণ।

সমাধান: আমরা রাউটারে Email Tool কনফিগার করবো এবং System > Scheduler ব্যবহার করে একটি Script তৈরি করবো। স্ক্রিপ্টটি প্রতিদিন নির্দিষ্ট সময়ে রাউটারের একটি .backup ফাইল তৈরি করবে এবং সেই ফাইলটি Attachment হিসেবে আপনার জিমেইল অ্যাকাউন্টে পাঠিয়ে দেবে। ফলে আপনার রাউটারের কনফিগারেশন সবসময় ক্লাউডে সুরক্ষিত থাকবে।

১. কনফিগারেশনের পূর্বশর্ত (Prerequisites)

MikroTik রাউটার থেকে ইমেইল পাঠানোর জন্য আমাদের একটি SMTP (Simple Mail Transfer Protocol) সার্ভার প্রয়োজন। আমরা যেহেতু Gmail ব্যবহার করবো, তাই আমাদের কিছু নির্দিষ্ট ডেটা লাগবে:

  • SMTP Server: smtp.gmail.com
  • Port: 587 (STARTTLS এর জন্য)
  • TLS: yes (সিকিউরিটির জন্য এটি বাধ্যতামূলক)
  • App Password: বর্তমানে জিমেইল তাদের রেগুলার পাসওয়ার্ড দিয়ে অন্য কোনো অ্যাপ থেকে লগইন করতে দেয় না। আপনাকে আপনার Google অ্যাকাউন্টের সেটিংসে গিয়ে (Security > 2-Step Verification > App passwords) "App Password" তৈরি করে নিতে হবে। এটি একটি ১৬-ডিজিটের বিশেষ পাসওয়ার্ড, যা শুধু আপনার রাউটার ব্যবহার করবে।

২. Email Tool কনফিগারেশন (Winbox)

এখন আমরা রাউটারে এই জিমেইলের তথ্যগুলো সেট করবো।

  • Winbox-এর বাম দিকের মেনু থেকে Tools > Email এ ক্লিক করুন।
  • একটি নতুন উইন্ডো ওপেন হবে। এখানে নিচের তথ্যগুলো পূরণ করুন:
    • Server: smtp.gmail.com (রাউটার নিজে থেকেই এটি আইপিতে কনভার্ট করে নেবে, যদি ডিএনএস ঠিক থাকে)।
    • Port: 587
    • Start TLS: yes ড্রপডাউন থেকে সিলেক্ট করুন।
    • From: আপনার জিমেইল অ্যাড্রেসটি দিন (যেমন: your_email@gmail.com)। এটি ইমেইলের "Sender" হিসেবে দেখাবে।
    • User: আপনার সম্পূর্ণ জিমেইল অ্যাড্রেসটি আবার দিন (যেমন: your_email@gmail.com)।
    • Password: আপনার তৈরি করা App Password টি এখানে দিন (রেগুলার পাসওয়ার্ড দেবেন না)।
  • Apply এবং OK দিন।

৩. Command Line (CLI) ব্যবহার করে কনফিগারেশন

টার্মিনাল থেকে এই কনফিগারেশনটি করা আরও দ্রুত এবং সহজ:

[admin@MikroTik] > /tool e-mail set server=smtp.gmail.com port=587 start-tls=yes from=your_email@gmail.com user=your_email@gmail.com password="your_16_digit_app_password"

৪. টেস্ট ইমেইল পাঠানো (Sending a Test Email)

কনফিগারেশন ঠিকমতো হয়েছে কি না, তা যাচাই করার জন্য আমরা একটি ম্যানুয়াল টেস্ট ইমেইল পাঠাবো।

Winbox থেকে আবার Tools > Email এ যান এবং ডান দিকের Send Email বাটনে ক্লিক করুন।

  • To: যে ইমেইলে পাঠাতে চান (আপনার নিজের বা অন্য কোনো ইমেইল)।
  • Subject: Test Email from MikroTik
  • Body: Hello! This is a test email sent from RouterOS.
  • সবশেষে Send Email বাটনে ক্লিক করুন। ইনবক্স চেক করে দেখুন ইমেইলটি এসেছে কি না।

CLI কমান্ড দিয়ে টেস্ট ইমেইল:

[admin@MikroTik] > /tool e-mail send to="recipient@example.com" subject="Test Email from MikroTik" body="Hello! This is a test email sent from RouterOS."

৫. Advanced Scenario: Script দিয়ে অটোমেটিক ব্যাকআপ পাঠানো

এখন আমরা সেই ম্যাজিক স্ক্রিপ্টটি তৈরি করবো, যা ব্যাকআপ ফাইল তৈরি করে ইমেইলে পাঠাবে।

  • Winbox থেকে System > Scripts এ যান এবং + (Add) আইকনে ক্লিক করুন।
  • Name: Auto-Backup-Email দিন।
  • নিচের বক্সে এই স্ক্রিপ্টটি কপি করে পেস্ট করুন (আপনার ইমেইল অ্যাড্রেসটি ঠিক করে নেবেন):
:local routerName [/system identity get name];
:local date [/system clock get date];
:local time [/system clock get time];
:local filename ($routerName . "-" . $date . ".backup");

# Create Backup
/system backup save name=$filename;

# Delay for 5 seconds to let the backup file finish writing
:delay 5s;

# Send Email with Attachment
/tool e-mail send to="your_email@gmail.com" subject="Backup: $routerName ($date)" body="Automated daily backup attached. Time: $time" file=$filename;

ম্যাজিক: এই স্ক্রিপ্টটি রান করলে রাউটার প্রথমে একটি ব্যাকআপ তৈরি করবে এবং ৫ সেকেন্ড পর সেই ব্যাকআপ ফাইলটি আপনার ইমেইলে অ্যাটাচমেন্ট (Attachment) হিসেবে পাঠিয়ে দেবে। এটি অটোমেটিক করার জন্য System > Scheduler এ গিয়ে এই স্ক্রিপ্টটির নাম দিয়ে প্রতিদিন রাত ২টায় রান করার শিডিউল তৈরি করে দিতে পারেন।

৬. Troubleshooting Tips (সাধারণ সমস্যা ও সমাধান)

সমস্যা (Problem) কারণ এবং সমাধান (Cause & Solution)
Error resolving server address আপনার রাউটার smtp.gmail.com কে আইপিতে (IP) রূপান্তর করতে পারছে না। রাউটারের IP > DNS সেটিংসে গিয়ে গুগলের ডিএনএস (8.8.8.8) বসানো আছে কি না চেক করুন এবং Allow Remote Requests এ টিক দিন।
Authentication failed / Invalid Username or Password আপনি জিমেইলের রেগুলার পাসওয়ার্ড ব্যবহার করছেন। Google এখন থার্ড-পার্টি অ্যাপের জন্য রেগুলার পাসওয়ার্ড সাপোর্ট করে না। আপনাকে অবশ্যই Google অ্যাকাউন্টের সিকিউরিটি সেটিংসে গিয়ে App Password তৈরি করে সেটি ব্যবহার করতে হবে।
Connection Timeout আপনার ইন্টারনেট সার্ভিস প্রোভাইডার (ISP) হয়তো স্প্যামিং কমানোর জন্য পোর্ট 587 ব্লক করে রেখেছে। এর সমাধান হিসেবে পোর্ট 465 ব্যবহার করে দেখুন এবং TLS এর মান yes এর বদলে tls-only ব্যবহার করতে পারেন।
File attached but email not sending যদি ব্যাকআপ ফাইলের সাইজ অনেক বড় হয়, তবে সেটি তৈরি হতে সময় লাগে। স্ক্রিপ্টে ফাইল সেভ করার পর :delay 10s (১০ সেকেন্ড) বিরতি ব্যবহার করুন, যাতে ফাইলটি সম্পূর্ণ রাইট (Write) হওয়ার পর ইমেইল টুল সেটি অ্যাটাচ করার সুযোগ পায়।

উপসংহার (Conclusion)

MikroTik RouterOS-এর Email Tool শুধুমাত্র একটি সাধারণ ফিচার নয়, বরং একটি শক্তিশালী মনিটরিং এবং অটোমেশন অস্ত্র। Netwatch বা Scripts এর সাথে ইমেইল টুলকে যুক্ত করে আপনি আপনার নেটওয়ার্ককে পুরোপুরি প্রোঅ্যাকটিভ (Proactive) করে তুলতে পারেন। রাউটার ডাউন হলে, ইউজার কানেক্ট হলে বা ব্যাকআপ তৈরি হলে—সবকিছুই এখন সরাসরি আপনার জিমেইল ইনবক্সে চলে আসবে, যা আপনার সময় বাঁচানোর পাশাপাশি নেটওয়ার্ক ম্যানেজমেন্টকে অনেক বেশি প্রফেশনাল করে তুলবে।

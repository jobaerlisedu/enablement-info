Connecting to the MikroTik Router via Winbox on MAC and IP addresses

30 May, 2026 Networking with MikroTik Jobaer Hossain

MikroTik (মাইক্রোটিক) রাউটার কনফিগারেশনের কথা উঠলেই সবার আগে যে টুলটির নাম মাথায় আসে, তা হলো Winbox। কমান্ড লাইন ইন্টারফেস (CLI) এর জটিলতা এড়িয়ে গ্রাফিক্যাল ইউজার ইন্টারফেসের (GUI) মাধ্যমে রাউটারকে সহজেই নিয়ন্ত্রণ করার জন্য এটি তৈরি করা হয়েছে।

একটি MikroTik রাউটারে লগইন বা কানেক্ট করার দুটি প্রধান পদ্ধতি রয়েছে: MAC Address এবং IP Address। নতুন একটি রাউটার বক্স থেকে বের করা থেকে শুরু করে একটি লাইভ ডেটা সেন্টারের রাউটার ম্যানেজ করা পর্যন্ত—এই দুটি পদ্ধতির কনসেপ্ট ক্লিয়ার থাকা একজন নেটওয়ার্ক ইঞ্জিনিয়ারের জন্য অত্যন্ত জরুরি। আজকের এই ডিপ-ডাইভ টিউটোরিয়ালে আমরা এই দুটি পদ্ধতি নিয়ে বিস্তারিত আলোচনা করবো।

Winbox কী? (What is Winbox?)

Winbox হলো MikroTik এর নিজস্ব একটি পোর্টেবল ইউটিলিটি সফটওয়্যার (যা ইন্সটল করার প্রয়োজন হয় না)। এর সাহায্যে আপনি RouterOS এর সকল ফিচার খুব সহজেই উইন্ডোজ অপারেটিং সিস্টেমে মাউস ক্লিকে কনফিগার করতে পারেন। এটি অত্যন্ত লাইটওয়েট এবং সিকিউর একটি সফটওয়্যার, যা MikroTik এর অফিসিয়াল ওয়েবসাইট থেকে ফ্রি ডাউনলোড করা যায়।

১. MAC Address দিয়ে কানেক্ট করার পদ্ধতি (Layer 2 Connection)

যখন আপনি একটি নতুন MikroTik রাউটার বক্স থেকে বের করেন অথবা কোনো রাউটার ফ্যাক্টরি রিসেট (Factory Reset) করেন, তখন তাতে কোনো কনফিগারেশন বা IP Address থাকে না। এমন পরিস্থিতিতে IP দিয়ে লগইন করা সম্ভব নয়। এখানেই MikroTik এর সবচেয়ে জাদুকরী ফিচারটি কাজে আসে—MAC Address Login।

  • ফিজিক্যাল কানেকশন: রাউটারের যেকোনো একটি ল্যান (LAN) পোর্টের সাথে আপনার ল্যাপটপ বা পিসির একটি নেটওয়ার্ক ক্যাবল (UTP) যুক্ত করুন।
  • Winbox ওপেন করুন: আপনার পিসিতে ডাউনলোড করা Winbox.exe ফাইলটি ডাবল ক্লিক করে ওপেন করুন।
  • Neighbors ট্যাব: Winbox এর নিচের দিকে Neighbors নামক একটি ট্যাব পাবেন। এখানে ক্লিক করলে Winbox ব্রডকাস্ট (Broadcast) করে খুঁজে বের করবে আপনার পিসির সাথে সরাসরি যুক্ত থাকা MikroTik রাউটারগুলো।
  • MAC Address সিলেক্ট করা: লিস্টে আপনার রাউটারটি দেখতে পেলে, তার MAC Address এর ওপর ক্লিক করুন। লক্ষ করবেন, ওপরের Connect To: বক্সে MAC Address টি বসে গেছে। (ভুল করে IP Address এর ওপর ক্লিক করবেন না)।
  • লগইন: নতুন রাউটারের ডিফল্ট Login: admin এবং Password: (খালি বা ব্ল্যাঙ্ক) রেখে Connect বাটনে ক্লিক করুন।

পেছনের মেকানিজম: MAC Address দিয়ে কানেকশনটি OSI Model এর Data Link Layer (Layer 2) এ কাজ করে। এটি রাউটার এবং পিসির মধ্যে একটি ব্রডকাস্ট চ্যানেল তৈরি করে ডাটা আদান-প্রদান করে।

২. IP Address দিয়ে কানেক্ট করার পদ্ধতি (Layer 3 Connection)

MAC Address দিয়ে লগইন করা সুবিধাজনক হলেও এটি খুব একটা স্থিতিশীল (Stable) নয়। রাউটারে যখন প্রচুর ট্রাফিক থাকে, তখন লেয়ার ২ ব্রডকাস্ট ড্রপ হতে পারে এবং Winbox সেশন ডিসকানেক্ট হয়ে যেতে পারে। তাই রাউটারে বেসিক কনফিগারেশন করার পর সব সময় IP Address দিয়ে লগইন করাটা হলো প্রফেশনাল প্র্যাকটিস।

  • IP অ্যাসাইন করা: রাউটারে লগইন করার জন্য প্রথমেই রাউটারের কোনো একটি ইন্টারফেসে একটি IP Address থাকতে হবে এবং আপনার পিসিকেও একই ব্লকের (Subnet) আইপি পেতে হবে। (যদি রাউটারে ডিফল্ট কনফিগারেশন থাকে, তবে রাউটারের আইপি হবে 192.168.88.1)।
  • Winbox ওপেন করুন: পুনরায় Winbox ওপেন করে Neighbors ট্যাবে যান।
  • IP Address সিলেক্ট করা: এবার লিস্টে রাউটারটি দেখতে পেলে, তার IP Address এর ওপর ক্লিক করুন। ওপরের Connect To: বক্সে আইপিটি চলে আসবে। অথবা, আপনি চাইলে সরাসরি Connect To: বক্সে রাউটারের আইপি টাইপ করতে পারেন (রিমোট লগইনের ক্ষেত্রে)।
  • লগইন: সঠিক ইউজারনেম এবং পাসওয়ার্ড দিয়ে Connect বাটনে ক্লিক করুন।

পেছনের মেকানিজম: IP Address দিয়ে কানেকশনটি OSI Model এর Network Layer (Layer 3) এবং Transport Layer এর TCP Port 8291 ব্যবহার করে কাজ করে। এটি অনেক বেশি স্থিতিশীল এবং রাউটেবল (Routable), অর্থাৎ আপনি ইন্টারনেটের মাধ্যমে পৃথিবীর যেকোনো জায়গা থেকেও আইপি দিয়ে লগইন করতে পারবেন।

৩. MAC vs IP Connection: কোনটি কখন ব্যবহার করবেন?

বৈশিষ্ট্য (Feature) MAC Address Connection IP Address Connection
OSI Layer Layer 2 (Data Link Layer) Layer 3 (Network Layer)
ব্যবহারের ক্ষেত্র নতুন রাউটার সেটআপ, আইপি ভুলে গেলে বা নেটওয়ার্ক কনফিগারেশন রিসেট হলে। রেগুলার অ্যাডমিনিস্ট্রেশন, রিমোট ম্যানেজমেন্ট এবং লাইভ ট্রাবলশুটিংয়ের জন্য।
স্থিতিশীলতা (Stability) কম। ট্রাফিক লোড পড়লে কানেকশন ড্রপ করতে পারে। সর্বোচ্চ। কানেকশন অত্যন্ত স্থিতিশীল থাকে।
রাউটিং (Routability) ল্যান (LAN) পার হয়ে অন্য নেটওয়ার্কে যেতে পারে না। সরাসরি যুক্ত থাকতে হবে। ইন্টারনেট বা রাউটারের মাধ্যমে রিমোটলি যুক্ত হওয়া যায়।

৪. Basic Configuration (CLI Command)

আপনি যদি প্রথমবার MAC Address দিয়ে লগইন করেন, তবে পরবর্তী সময়ে IP দিয়ে লগইন করার জন্য আপনাকে ইন্টারফেসে একটি আইপি বসাতে হবে। Winbox এর New Terminal ওপেন করে নিচের কমান্ডটি দিন:

[admin@MikroTik] > ip address add address=192.168.1.1/24 interface=ether2

এখন আপনার পিসিতে 192.168.1.2 আইপিটি বসিয়ে খুব সহজেই Winbox এ 192.168.1.1 টাইপ করে লেয়ার ৩ কানেকশন তৈরি করতে পারবেন।

৫. Troubleshooting Tips (সমস্যা ও সমাধান)

Winbox দিয়ে কানেক্ট করতে গিয়ে কিছু কমন সমস্যা হতে পারে, যার সমাধান নিচে দেওয়া হলো:

  • Winbox এর Neighbors লিস্টে রাউটার শো করছে না: এটি সবচেয়ে বেশি হয় Windows Firewall বা অ্যান্টিভাইরাসের কারণে। আপনার উইন্ডোজের ফায়ারওয়াল সাময়িকভাবে বন্ধ (Turn Off) করে Winbox রিস্টার্ট দিন। এছাড়া পিসিতে একাধিক নেটওয়ার্ক এডাপ্টার (যেমন VirtualBox বা VMWare এর এডাপ্টার) চালু থাকলে সেগুলো ডিসেবল করে দিন।
  • MAC দিয়ে কানেক্ট করার কিছুক্ষণ পরই ডিসকানেক্ট হয়ে যাচ্ছে: এটি Layer 2 ব্রডকাস্ট ট্রাফিকের কারণে হয়। দ্রুত IP > Addresses মেনুতে গিয়ে একটি আইপি বসিয়ে নিন এবং এরপর IP Address দিয়ে লগইন করে কাজ করুন।
  • Wrong Username or Password: যদি পাসওয়ার্ড কাজ না করে, তবে ফিজিক্যাল রাউটারের পাওয়ার অফ করে Reset বাটন চেপে ধরে পাওয়ার অন করুন। আলো ফ্ল্যাশ (Blink) করলে ছেড়ে দিন। এরপর MAC দিয়ে admin ইউজারনেম দিয়ে বিনা পাসওয়ার্ডে লগইন করতে পারবেন।

উপসংহার (Conclusion)

Winbox হলো MikroTik সিস্টেমের মেরুদণ্ড এবং এর MAC Address লগইন ফিচারটি নেটওয়ার্ক ইঞ্জিনিয়ারদের জন্য একটি বড় আশীর্বাদ। যেখানে অন্যান্য ব্যান্ডের রাউটারে আইপি কনফিগারেশন ভুল হলে সরাসরি কনসোল ক্যাবলের (Console Cable) ওপর নির্ভর করতে হয়, সেখানে MikroTik এ আপনি ইথারনেট ক্যাবল দিয়েই লেয়ার ২ তে লগইন করে সমস্যা সমাধান করতে পারেন। তবে প্রোডাকশন এনভায়রনমেন্টে সব সময় IP Address ব্যবহার করে লগইন করাই সবচেয়ে নিরাপদ এবং স্থিতিশীল পদ্ধতি।

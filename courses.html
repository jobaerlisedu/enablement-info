Foundation of Networking

নেটওয়ার্কিংয়ের মৌলিক ধারণা থেকে অ্যাডভান্স লেভেল পর্যন্ত একটি কমপ্লিট থিওরিটিক্যাল গাইড, যা আপনার টেকনিক্যাল ক্যারিয়ারের ভিত্তি গড়ে তুলতে সাহায্য করবে।

কোর্সটি শুরু করুন

MikroTik Certified Network Associate (MTCNA)

মাইক্রোটিক রাউটার কনফিগারেশন, ব্যান্ডউইথ ম্যানেজমেন্ট এবং সিকিউরিটি নিশ্চিত করার একটি পূর্ণাঙ্গ হ্যান্ডস-অন কোর্স।

কোর্সটি শুরু করুন

Cisco Certified Network Associate (CCNA 200-301)

সিসিএনএ (CCNA) লেভেল থেকে শুরু করে অ্যাডভান্স রাউটিং এবং সুইচিং শিখুন আমাদের এই কমপ্লিট সিসকো ট্রেনিং মডিউলে।

কোর্সটি শুরু করুন

Windows Server Administration

উইন্ডোজ সার্ভার অ্যাডমিনিস্ট্রেশন, অ্যাক্টিভ ডিরেক্টরি এবং নেটওয়ার্ক সিকিউরিটি শিখুন আমাদের এই কমপ্লিট উইন্ডোজ সার্ভার ট্রেনিং মডিউলে।

কোর্সটি শুরু করুন

Linux Server Administration

লিনাক্স সার্ভার অ্যাডমিনিস্ট্রেশন, কমান্ড লাইন, ফাইল সিস্টেম এবং সার্ভার কনফিগারেশন শিখুন আমাদের এই কমপ্লিট লিনাক্স ট্রেনিং মডিউলে।
